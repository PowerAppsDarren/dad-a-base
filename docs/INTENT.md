# Project Intent

## What This IS

A gloriously over-engineered dad joke delivery system that doubles as a Docker Compose multi-service orchestration showcase. It's a full-stack app (FastAPI + React + PostgreSQL + Nginx) deployed with a single `docker-compose up` command.

## What This IS NOT

- Not a production joke service — it's a learning/demo project
- Not an AI joke generator — all jokes are human-curated
- Not a social platform — no accounts, no sharing
- Not a mobile app — web only

## Why This Exists

To demonstrate enterprise-grade infrastructure patterns (health checks, reverse proxy, schema design, service orchestration) in a fun, approachable context. If you're going to learn Docker Compose, you might as well laugh while doing it.

## Target User

Developers learning Docker Compose, full-stack architecture, or anyone who needs a dad joke delivered with unnecessary infrastructure.

## Core Principles

1. **Comedy over complexity** — the architecture jokes are as important as the code
2. **One-command deploy** — `docker-compose up --build` and you're laughing
3. **Educational showcase** — every infrastructure choice is intentional and documented
